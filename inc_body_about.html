RipTextAudio.com Motivations

Make you reading life easier! We deliver and convert text to audio for FREE! The steps are easy as 1,2,3:

1. User paste text.
2. Click to generated the audio.
3. Listen or download the audio files (zipped or individual files).

Philip Jahmani Chauvet

For many years, Philip Jahmani Chauvet, has been using Java to do audio and text processing. He had a Java utility and tool that would take text and convert it to audio. It was a pet project on his desktop that really made learning a bit more easier! Several people used his tool and recommended that he create a real application, or make it available on the Internet.

Being a proponent of Open Source Software, he created RipTextAudio.com. It's features are Free!

Passion for Java and GWT

Philip Jahmani Chauvet has a great passion for technology, his love for Java and the Google Web Toolkit (GWT) from Google. RipTextAudio.com is using Java and GWT to deliver audio file(s) from text online. This website was created using GWT!
